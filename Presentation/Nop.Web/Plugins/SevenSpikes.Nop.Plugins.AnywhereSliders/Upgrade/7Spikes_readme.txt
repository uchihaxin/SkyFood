Steps:
1. Backup your existing database
2. If you haven't upgraded your nopCommerce website to version 3.5 already, then follow the instructions in the "nopCommerce_readme.txt".
3. Copy this plugin and the "SevenSpikes.Core" plugin folders to the Plugins folder of nopCommerce.
4. Start your site. If you have made any changes to this plugin in 3.4 then you need to merge them in the new version of the plugin.
